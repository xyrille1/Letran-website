# **App Name**: Letran Manaoag

## Core Features:

- Event Calendar: Display upcoming school events and important dates.
- Admissions Information: Provide details about enrollment procedures and requirements.
- Campus Tour: Allow a generative AI model to decide how to incorporate location information for places on campus using a 'tool' and construct a tour route.
- Interactive Map: Display a detailed campus map with building information and points of interest.
- School News: Display announcements.
- Contact Information: List contact details for key departments and personnel.

## Style Guidelines:

- Primary color: Letran Gold (#D4AF37) to reflect the school's identity and tradition.
- Background color: Desaturated light beige (#F5F5DC) to provide a warm, inviting feel.
- Accent color: Deep Brown (#8B4513), for buttons and important UI elements, creating contrast and focus.
- Body font: 'Inter', sans-serif, for a clean and modern reading experience.
- Headline font: 'Cinzel', serif, for titles and headings.
- Use Font Awesome icons, styled in Letran Gold.
- Use clear and consistent layouts.
- Subtle animations on hover to improve user engagement.
